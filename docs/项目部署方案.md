# AI Canvas 部署方案

## 一、部署架构

```
┌─────────────────┐     ┌─────────────────────┐     ┌─────────────┐
│  Chrome 浏览器   │────▶│  阿里云 ECS (API)    │────▶│  Notion API │
│  (GitHub Release)│     │  api.example.com     │     └─────────────┘
└─────────────────┘     └─────────────────────┘
```

**分发方式：** 扩展通过 GitHub Release 发布，用户下载后手动加载到 Chrome

## 三、后端 API 部署（阿里云）

### 3.1 购买 ECS 实例

1. 登录 [阿里云控制台](https://ecs.console.aliyun.com/)
2. 创建 ECS 实例：
   - 地域：华东1（杭州）或就近选择
   - 规格：ecs.t6-c1m1.large（1核2G）足够
   - 操作系统：Ubuntu 22.04 LTS
   - 存储：40G SSD
   - 安全组：开放 80、443、22 端口

### 3.2 服务器环境配置

```bash
# SSH 连接到服务器
ssh root@你的服务器IP

# 更新系统
apt update && apt upgrade -y

# 安装 Node.js 20.x
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
apt install -y nodejs

# 安装 pnpm
npm install -g pnpm

# 安装 PM2（进程管理）
pnpm install -g pm2

# 安装 Nginx
apt install -y nginx

# 安装 Certbot（SSL 证书，如果有域名）
apt install -y certbot python3-certbot-nginx
```

### 3.3 部署 API 代码

```bash
# 创建项目目录
mkdir -p /var/www/ai-canvas-api
cd /var/www/ai-canvas-api

# 克隆代码
git clone https://github.com/liushuai-man/AI-Canvas.git .

# 进入 API 目录
cd api

# 安装依赖
pnpm install

# 编译 TypeScript
pnpm build
# 如果没有 build 脚本，添加到 package.json：
# "build": "tsc"

# 创建环境变量文件
cat > .env << 'EOF'
NOTION_TOKEN=你的notion_token
PORT=8080
NOTION_CLIENT_ID=你的client_id
NOTION_CLIENT_SECRET=你的client_secret
NOTION_REDIRECT_URI=http://120.55.2.225/api/notion/callback
SESSION_SECRET=随机生成的密钥字符串
EOF
```

### 3.4 修改 API 代码支持生产环境

**添加 CORS 支持：**

```bash
# 安装 cors
pnpm add cors @types/cors
```

修改 `api/src/app.ts`：

```typescript
import express from 'express';
import cors from 'cors';
import { env } from './config/env';
import notionRoutes from './routes/notion';

const app = express();

// CORS 配置
app.use(
  cors({
    origin: '*', // Chrome 扩展的 origin 是 chrome-extension://xxx
    methods: ['GET', 'POST', 'PUT', 'DELETE'],
    allowedHeaders: ['Content-Type', 'Authorization'],
  })
);

app.use(express.json());
app.use('/api/notion', notionRoutes);

app.listen(env.port, () => {
  console.log(`server running on http://0.0.0.0:${env.port}`);
});
```

### 3.5 PM2 启动服务

```bash
# 使用 PM2 启动（编译后的 JS）
pm2 start dist/app.js --name ai-canvas-api

# 设置开机自启
pm2 startup
pm2 save

# 查看日志
pm2 logs ai-canvas-api
```

### 3.6 Nginx 配置

```nginx
# /etc/nginx/sites-available/ai-canvas-api
server {
    listen 80;
    server_name _;

    location / {
        proxy_pass http://127.0.0.1:8080;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
    }
}
```

```bash
# 启用配置
ln -s /etc/nginx/sites-available/ai-canvas-api /etc/nginx/sites-enabled/
nginx -t
systemctl restart nginx
```

## 四、Chrome 扩展部署（GitHub Release）

### 4.1 修改 API 地址配置

修改 `extension/src/config/env.ts`，将生产环境地址改为你的服务器公网 IP：

```typescript
const isDev = import.meta.env.DEV;

export const API_BASE_URL = isDev
  ? 'http://localhost:8080'
  : 'http://120.55.2.225';
```

修改 `extension/wxt.config.ts`，添加你的服务器 IP：

```typescript
import { defineConfig } from 'wxt';

export default defineConfig({
  modules: ['@wxt-dev/module-react'],
  manifest: {
    permissions: ['storage', 'downloads'],
    host_permissions: ['http://localhost:8080/*', 'http://120.55.2.225/*'],
  },
});
```

### 4.3 构建生产版本

```bash
cd extension

# 构建 Chrome 扩展
pnpm build

# 构建 zip 包（用于 GitHub Release）
pnpm zip
```

### 4.4 发布到 GitHub Release

1. **更新版本号**：修改 `extension/package.json` 中的 `version`

2. **创建 Git Tag**：

```bash
git tag -a v0.0.3 -m "Release v0.0.3"
git push origin v0.0.3
```

3. **在 GitHub 创建 Release**：
   - 访问 https://github.com/liushuai-man/AI-Canvas/releases
   - 点击「Create a new release」
   - 选择刚创建的 tag
   - 填写 Release 标题和说明
   - 上传 `.output/ai-canvas-0.0.3-chrome.zip` 作为附件
   - 点击「Publish release」

4. **Release 说明模板**：

```markdown
## AI Canvas v0.0.3

### 功能

- 从 ChatGPT 对话提取内容
- 可视化画布编辑
- 导出为 PDF/MD/TXT/HTML
- 保存到 Notion

### 安装方法

1. 下载 `ai-canvas-0.0.3-chrome.zip`
2. 解压到本地文件夹
3. 打开 Chrome，访问 `chrome://extensions/`
4. 开启「开发者模式」
5. 点击「加载已解压的扩展程序」
6. 选择解压后的文件夹

### 注意事项

- 需要连接到 AI Canvas 后端服务
- 首次使用需要登录 Notion 账户
```

## 七、常见问题

### Q1: Chrome 扩展无法访问 API

**原因：** CORS 问题或 `host_permissions` 未配置
**解决：** 确保 `wxt.config.ts` 中已添加服务器 IP 到 `host_permissions`

### Q2: Notion OAuth 回调失败

**原因：** 回调地址与 Notion 配置不一致
**解决：** 确保 Notion 集成的 Redirect URIs 与 `.env` 中的 `NOTION_REDIRECT_URI` 一致

### Q3: API 服务器内存不足

**解决：** 添加 swap 分区

```bash
fallocate -l 2G /swapfile
chmod 600 /swapfile
mkswap /swapfile
swapon /swapfile
echo '/swapfile none swap sw 0 0' >> /etc/fstab
```

### Q4: 用户安装扩展后无法使用

**原因：** 可能是 API 地址配置错误
**解决：** 检查 `extension/src/config/env.ts` 中的 `API_BASE_URL` 是否正确

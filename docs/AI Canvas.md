# AI画布浏览器插件产品文档

## 一、产品概述

### 1.1 产品名称

AI画布（AI Canvas）

### 1.2 产品定位

AI画布是一款基于React+TS+WXT框架开发的浏览器插件，核心定位为「AI对话结果整理与导出工具」，专注解决用户与AI对话后，结果难以灵活编辑、布局调整不便、导出格式单一的痛点，为需要记录、整理AI对话成果的用户，提供“导入-编辑-布局-导出”一站式服务，让AI对话内容可定制、可留存、可复用。

### 1.3 核心价值

- 高效整理：一键导入AI对话全流程内容，无需手动复制粘贴，节省整理时间；

- 灵活编辑：支持对话内容的增删改查，可自由调整布局，适配不同使用场景的呈现需求；

- 多格式导出：支持PDF、Markdown等多种常用格式导出，满足用户存档、分享、编辑的多样化需求；

- 轻量便捷：浏览器插件形态，无需切换应用，在AI对话页面即可完成全流程操作，降低使用门槛。

### 1.4 适用场景

针对用户与AI（如ChatGPT、Gemini、Claude等各类AI对话工具）结束对话后，需要对对话过程和结果进行整理、优化、留存的场景，具体包括：

- 学生场景：整理AI答疑的知识点、解题思路，调整内容顺序，导出为笔记存档，方便后续复习；

- 教师场景：汇总AI生成的教学案例、课件素材，编辑优化后导出，用于课堂展示或备课留存；

- 其他场景：科研人员整理AI辅助分析的结果、职场人整理AI生成的方案草稿，通过布局调整和导出，提升内容复用性。

**4.1 技术栈**

• 前端框架：React（18+）+ TypeScript，确保代码可维护性和类型安全；

• 插件框架：WXT（最新稳定版），简化浏览器插件开发流程，自动处理Manifest V3配置、跨浏览器兼容等问题；

• 核心依赖库：
    

￮ React-DnD：实现内容模块拖拽排序；

￮ html2pdf.js：实现PDF格式导出；

￮ Marked-it：实现Markdown格式导出；

￮ zustand：轻量级状态管理，管理画布编辑状态、布局状态等。

• 兼容性：支持Chrome、Edge、Firefox主流浏览器，基于Manifest V3标准开发，确保插件稳定性和安全性。

**4.3 模块分工（基于WXT）**

• Popup模块：作为插件的主交互界面，承载画布编辑、布局调整、导出操作等核心功能，由React+TS实现；

• Content Script模块：注入到AI对话页面，监听分享事件、抓取对话内容，将内容传递至Popup模块；

• Background模块：常驻浏览器后台，负责本地存储（用户编辑内容、布局设置）、跨模块消息通信、导出文件的后台处理，确保数据持久化和功能稳定性。

## 五、用户操作流程

**5.1 整体流程**

安装插件 → 打开AI对话页面完成对话 → 导入对话内容至画布 → 编辑内容/调整布局 → 导出文件 → 完成留存/分享

**5.2 详细步骤**

\1. 插件安装：用户在浏览器应用商店（Chrome网上应用店、Edge附加组件商店等）搜索“AI画布”，点击安装，安装完成后，浏览器右上角显示插件图标；

\2. 对话准备：用户在任意AI对话平台（如ChatGPT）完成对话，确认需要整理的对话内容；

\3. 内容导入：
    

￮ 方式一：点击AI对话页面的“分享”按钮，选择“分享至AI画布”，插件自动抓取对话内容并导入画布；

￮ 方式二：复制AI对话内容，点击浏览器右上角插件图标打开Popup，粘贴至导入框，点击“导入”完成内容加载。

\4. 内容编辑：双击画布上的文本进行修改，点击删除按钮删除冗余内容，使用工具栏添加新内容、调整格式，通过搜索框快速定位内容；

\5. 布局调整：鼠标拖拽内容模块调整顺序，点击“分栏”按钮设置并列布局，选中多个模块点击“分组”进行分类管理；

\6. 文件导出：点击Popup右上角“导出”按钮，选择目标格式（PDF/Markdown），设置导出参数（可选），点击“确认”，文件自动下载至本地；

\7. 后续操作：用户可再次打开插件，查看、编辑之前保存的画布内容，或导入新的AI对话内容进行整理。
